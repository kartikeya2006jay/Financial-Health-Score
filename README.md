# Financial Health Score

An AI/ML-driven MSME Financial Health Card application designed to aggregate alternate data (GST, UPI, AA, EPFO) and compute a multidimensional financial health score for New-to-Credit (NTC) and New-to-Bank (NTB) enterprises.

## Tech Stack
- **Frontend**: React, Vite, Lucide Icons
- **Backend**: FastAPI (Python)
- **Styling**: Custom CSS with Glassmorphism aesthetic

## Features
- Dynamic form to input enterprise alternate data identifiers (GSTIN, UPI VPA, AA Consent ID, EPFO Reg No).
- Compute multidimensional health scores (Overall, Cash Flow, GST, EPFO, UPI).
- Beautiful, fully responsive glassmorphic UI dashboard to visualize risk and strength metrics.
- Actionable credit insights generated via a mocked ML inference engine.
